## 📜 三次握手流程图
```text
客户端（Client）                                服务端（Server）

      SYN = 1, Seq = x
  ------------------------->       第一次握手：Client 发起连接请求

                            SYN = 1, ACK = 1, Seq = y, Ack = x+1
  <-------------------------       第二次握手：Server 同意连接请求并响应

      ACK = 1, Seq = x+1, Ack = y+1
  ------------------------->       第三次握手：Client 确认响应，连接建立

```
