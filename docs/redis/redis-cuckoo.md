
在前面讲布隆过滤器时候，我们谈到一个他的一个特点 不能被删除，这里我们看一下另外一种过滤器

## ✅ 布隆过滤器 vs 布谷鸟过滤器 对比表

| 特性             | 布隆过滤器         | 布谷鸟过滤器           |
|------------------|--------------------|-------------------------|
| 是否支持删除     | ❌ 不支持          | ✅ 支持                 |
| 插入效率         | 较高               | 高                      |
| 查询效率         | 高                 | 高                      |
| 空间占用         | 比较小             | 稍大（但比布隆+删除更优） |
| 误判率           | 存在误判（只误判“存在”） | 同样有误判         |
